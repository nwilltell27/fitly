# FitLy - Fitness Tracker
Welcome to **FitLy**, the fitness application designed to help users track exercise progress, set goals, manage nutrition, and achieve GREATNESS. 

## Technologies

- Django
- PostgreSQL
- HTML5

- Bootstrap
